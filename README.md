Work Scripts
============

Quick scripts and one liners.

1. Concatenate .csv files

Python script to convert, clean up and concatenate some old report files into one large .csv files.

2. Parse Edifact

Python scripts to read in edifact files and export the values in a more readable format.

3. DiffReports

Dos line to run multiple excel Report comparisons using [ExcelCompare](https://github.com/na-ka-na/ExcelCompare). Saves a bit of typing each time.
